```
	方法一
	for (var i = 0,len = rows.length; i < len; i++){
		if(i % 2){
			className = 'even',
		} else {
			className = 'old'
		}

			但此数为偶数，那么按位与运算的结果是0，
			如果为奇数  ，那么按位与运算的结果是1
	for (var i = 0,len = rows.length; i < len; i++){
		if(i & 1){
			className = 'old',
		} else {
			className = 'even'
		}
}
```
